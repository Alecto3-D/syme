What's going on here?

Syme makes it easy to set up disposable sessions for collaborating on GitHub projects via ssh and tmux.

  1. Enter the name of a GitHub repo. (Authorize Syme via GitHub if you haven't already.)
  2. Enter your AWS credentials and names of GitHub users to invite.
  3. Wait for your instance to boot.
  4. SSH into the instance using the command shown and launch `tmux`.
  5. Send the login info to the users you have invited.

Syme handles launching the instance, setting up public keys, installing basic packages, and cloning the repository in question.

Your AWS credentials are kept in an encrypted cookie in your browser and aren't stored server-side beyond the scope of your request.
